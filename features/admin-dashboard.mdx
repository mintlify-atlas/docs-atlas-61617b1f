The microfeed admin dashboard provides a complete GUI for managing your content, customizing your site, and monitoring your feed. If you've used WordPress, you'll find the interface familiar and intuitive.

## Accessing the Admin Dashboard

Your admin dashboard is located at:

```
https://your-project-name.pages.dev/admin/
```

<Warning>
The admin dashboard must be protected by Cloudflare Zero Trust. You'll configure this during the initial setup process.
</Warning>

## Admin Home

The home screen displays three key sections:

When you first access the admin dashboard, you'll see a checklist to complete the setup:

2. **Protected Admin Dashboard** - Secure with Cloudflare Zero Trust

### What's New

## Creating Content

### Add New Item Workflow

1. **Select Content Type**
   Choose from audio, video, image, document, blog post, or external URL

2. **Upload or Link Media**
   - Drag and drop files for upload
   - Enter URLs for external links
   - Files are automatically uploaded to Cloudflare R2

3. **Add Metadata**
   - **Title** (required)
   - **Description** with rich text editor
   - **Cover image** (optional)
   - **Publication date** with datetime picker
   - **Custom link** (optional external URL)

4. **Configure Podcast Settings** (for audio content)
   - Episode number
   - Season number
   - Episode type (full, trailer, bonus)
   - Explicit content flag

5. **Set Publication Status**
   - **Published:** Listed on web/RSS/JSON feeds, visible via direct link
   - **Unpublished:** Not listed, not visible
   - **Unlisted:** Not listed, but visible via direct link

6. **Save and Publish**

## Managing Items

The **All Items** page provides a list view of all your content:

- Filter by publication status
- Search by title
- Sort by publication date
- Quick edit and delete actions
- Batch operations

<Tip>
Changes to items are immediately reflected in your RSS, JSON, and web feeds.
</Tip>

## Edit Channel

Configure your feed's global settings:

### Basic Information
- **Title:** Your feed name
- **Description:** Rich HTML description of your feed
- **Publisher:** Author/creator name
- **Link:** Homepage URL
- **Language:** Content language code
- **Cover Image:** Feed artwork (recommended: 3000x3000px)

### Podcast Settings
- **iTunes Category:** Primary and subcategory
- **Podcast Type:** Episodic or serial
- **Explicit Content:** Flag for mature content
- **Copyright:** Copyright notice
- **Owner Email:** Contact email for podcast directories

### Advanced Settings
- **Block from iTunes:** Prevent indexing
- **Mark as Complete:** Indicate no new episodes
- **New Feed URL:** Redirect subscribers to new feed

## Settings Pages

### Subscribe Methods

Configure where listeners can subscribe to your feed:

- Apple Podcasts
- Spotify
- Google Podcasts
- Amazon Music
- Overcast
- Pocket Casts
- RSS
- JSON
- Custom platforms

Each subscribe method can be enabled/disabled and customized with your specific URL.

### Tracking URLs

Set up third-party tracking for analytics:

```
https://op3.dev/e/
https://dts.podtrac.com/redirect.mp3/
```

Tracking URLs are automatically prepended to media file URLs in RSS and JSON feeds.

<Note>
This is a common practice in the podcast industry for monitoring download statistics.
</Note>

### Access Control

Control who can view your feed:

- **Public:** Entire site publicly accessible
- **Offline:** Site returns 404 for all public pages
- **Passcode:** Protect all pages with a password

```javascript
export const CHANNEL_STATUSES = {
  PUBLIC: 'public',
  OFFLINE: 'offline',
  PASSCODE: 'passcode',
};
```

### Custom Code

Edit HTML/CSS templates directly in the browser. See [Custom Theming](/features/custom-theming) for details.

### API Settings

Configure your JSON feed API:

- Default items per page
- Maximum items per page
- Default sort order (newest first or oldest first)

## Admin Interface Architecture

The admin dashboard is built with React and uses client-side rendering:

```javascript
// Admin apps structure
- ClientAdminHomeApp       // Dashboard home
- ClientAdminItemsApp      // Content management
- ClientAdminChannelApp    // Channel settings
- ClientAdminSettingsApp   // Global settings
- ClientAdminCustomCodeEditorApp  // Theme editor
```
